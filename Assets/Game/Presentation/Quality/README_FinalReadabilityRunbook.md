# Final Readability / Accessibility Runbook

## Formål
Kjøre siste sign-off uten å lyve i bundle.

## Må verifiseres i runtime
1. Tile kan identifiseres på under 1 sekund
2. Cascade depth 3+ er fortsatt lesbar
3. Effekter forklarer state, ikke skjuler den
4. HUD er sekundær til board
5. Audio støtter feedback uten å dominere

## Kritisk regel
Ingen subjektiv PASS skal skrives automatisk uten runtime capture og menneskelig vurdering.
