# Nature Light Art Pipeline

Dette er Phase B art-pipeline for endelig authored Nature Light visuals.

## Mål
- Erstatte placeholder visuals med authored tile art
- Holde readability som øverste kontrakt
- Ikke introdusere alternativ gameplay-sannhet

## Leveranse per tile
- base
- symbol
- highlight

## Kvalitetskrav
- Tile må identifiseres på under 1 sekund
- Symbol og base må være distinkte ved cascade depth 3+
- Highlight skal forklare state, ikke skjule den

## Kritisk kontrakt
Presentation er projection av engine-truth, aldri alternativ sannhet.
