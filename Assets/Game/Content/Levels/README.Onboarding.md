# Onboarding Levels 1–5

Design intent:
- Level 1: Immediate fun, instant reward (<5s)
- Level 2: Teach specials safely
- Level 3: Teach blockers without punishment
- Level 4: Introduce light planning
- Level 5: First possible loss (mild + rewarded)

Rules:
- Never more than 10–15s without feedback
- First loss must still feel good
- If testers quit before Level 5 → onboarding failed
