## Subtitle / Short Description

## Description

BUXPuzzle is a readable, relaxed match-3 puzzle game built around calm clarity, tactile feedback, and deterministic board play. Swap tiles, resolve matches, and progress through endless boards designed to stay understandable during cascades.
